Projects

Medicine Search Engine (MSE)

Medicine Search Engine (MSE) is a web-based platform designed to make finding medicines fast and reliable during emergencies. Pharmacists can upload their medicine inventory into the system using a secure admin panel. Each entry includes details such as medicine name, brand, quantity and availability status. This data is stored in a MongoDB database, making it easy to update and manage in real time. On the user side, customers can search for medicines by name and instantly view which nearby pharmacies have them in stock. The system retrieves data dynamically from the database and displays it in a clean, user-friendly interface built with HTML, CSS and Flask. This ensures users can quickly find what they need without manually calling multiple pharmacies.

Pixel Perfect

Pixel Perfect is an interactive photography competition platform designed to manage submissions, enable student participation and ensure fair voting. The admin panel allows organizers to upload photos, assign unique numbers and track real-time votes. It also provides a dashboard view to monitor participation, identify which students have not voted and display leading entries. On the student side, participants can log in using their verified email, browse all submissions and vote for their favorite entry. The platform ensures one vote per student and updates results instantly. Built with HTML, CSS, Flask and MongoDB, Pixel Perfect creates an engaging experience by combining competition management and live result tracking in a seamless interface.

IPL Win Predictor

IPL Win Predictor is a machine learning-based application that predicts the probability of a team winning an IPL match based on live match data. Users can input details such as batting team, bowling team, current score, overs completed and wickets fallen. The model then calculates the winning probability in real-time, helping cricket fans analyze the match dynamically. The system is built using Python, Pandas and Scikit-learn, trained on historical IPL data to ensure accuracy. It provides instant, visually appealing results, making it a great tool for enthusiasts to evaluate match outcomes and enhance their cricket-watching experience.

Survey Project

Survey Project is a web-based platform designed to collect, store and analyze survey data efficiently. Admins can create custom surveys, upload them to the platform and share them with participants. Students or users can log in, submit their responses securely and ensure that each response is tracked to avoid duplication. The project uses HTML, CSS, JavaScript for the frontend, Flask for backend processing and MongoDB for storing all survey data. Admins can view real-time results, analyze response patterns and generate insights for decision-making. This makes it a simple yet powerful tool for conducting department-wide surveys, polls or feedback sessions.

Student & Tutor Web App

The Student and Tutor Web App is an interactive platform designed to seamlessly connect students with qualified tutors for personalized learning. Students can create profiles, browse tutors based on subjects or expertise and schedule sessions according to their convenience. The platform supports features like live video lessons, chat-based doubt solving and progress tracking, ensuring that learners receive a structured and engaging educational experience. Additionally, students can rate and provide feedback on tutors, helping maintain the quality of teaching and enabling informed choices for future learners. On the other side, tutors can create professional profiles highlighting their qualifications, areas of expertise and available time slots. They can manage their schedules, conduct live sessions and monitor student progress through analytics and performance reports. The app also integrates secure payment gateways, automated session reminders and learning resources, making it a comprehensive solution for both teaching and learning. By bridging the gap between students and educators, the platform promotes flexible, efficient and accessible education for everyone.
